### 说明

- 第一种订阅，会选取昨天所有订阅中时间最新的订阅；

- 第二种订阅，会遍历昨天所有的订阅文件，选取size最大的前5个订阅，生成订阅链接；链接最后面的`yb_v2rayse_sub1.yaml` 中的 `sub1` 可修改为 `sub2`、`sub3` 等，最多支持到数字5。
